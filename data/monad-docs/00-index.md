# Monad 기술 문서 요약 (RAG 참조용)

이 디렉토리는 Monad 공식 문서의 핵심 기술 사양을 요약한 텍스트 파일입니다.
RAG 엔진이 배포 에러 분석, 코드 수정 제안 시 이 문서를 참조할 수 있습니다.

## 문서 목록

1. **01-parallel-execution.md** - 병렬 실행, Optimistic Execution, State Conflict
2. **02-rpc-and-gas.md** - RPC 호환성, 가스 정책, Reserve Balance
3. **03-evm-and-pectra.md** - EVM 호환성, Opcode, Precompiles, 컨트랙트 크기
4. **04-consensus-and-mempool.md** - MonadBFT, Block, Finality, Mempool
